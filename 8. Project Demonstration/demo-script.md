# 8. Project Demonstration

## Demo Checklist
- [ ] `uvicorn main:app --reload` running locally, app open at http://127.0.0.1:8000
- [ ] Valid `GOOGLE_API_KEY` set in `.env`
- [ ] Record screen (or present live) walking through all 5 modules

## Suggested Demo Script (3–5 min)
1. **Intro (20s):** "This is EduGenie, a Google Gemini powered learning assistant built with
   FastAPI and a lightweight HTML/CSS frontend."
2. **Explain module (40s):** Enter "Pythagoras Theorem", level Beginner → show the generated
   explanation.
3. **Q&A module (30s):** Ask "Which is the largest ocean?" → show the instant answer with
   context (Scenario 1).
4. **Quiz module (40s):** Generate a 5-question quiz on "Pythagoras Theorem" → show the MCQs
   and answer key (Scenario 2).
5. **Summarization module (30s):** Paste a short paragraph → show the bullet-point summary.
6. **Learning Path module (40s):** Enter "SQL" → show the Beginner/Intermediate/Advanced
   roadmap (Scenario 3).
7. **Architecture recap (30s):** Briefly show the architecture diagram and explain how the
   FastAPI backend routes each request to its module.
8. **Close (10s):** Mention the GitHub repo link and tech stack used.
